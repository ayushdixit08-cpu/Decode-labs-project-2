# 💰 Expense Tracker (Python)

A simple command-line Expense Tracker built using Python. This project allows users to enter multiple expenses, calculates the total amount spent, and displays a summary at the end.

## 📌 Features

- Add multiple expenses
- Continuous input using a loop
- Running total after each expense
- Input validation using `try-except`
- Prevents negative expense values
- Exit anytime using `quit` or `exit`
- Displays:
  - Total number of expenses
  - Total amount spent
  - Average expense
